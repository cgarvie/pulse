Using Pulse is easy.

Install with pip

Simply run the code below in a terminal to install pulse. You probably want to run this in a virtualenv.

$ pip install pulseapp

Test it out

Run python and let's try pulse out.

$ python
>>> from pulseapp import Pulse
>>> pulse = Pulse('{{ mykey }}') # This is your api key, {{ current_user.email }}
>>> pulse.event('I made a sandwich', 'ham & cheese')
>>> pulse.event('I sent an email', 'invitation')
>>> pulse.event('attempted website login', 'wrong password')

That's it!

Once you've run pulse.event() at least once, refresh this page to view your dashboard.
